# Momento — main 分支

> ✅ 此分支为稳定主分支，用于保存已经验证、可继续长期维护的功能。改动应尽量保持可运行、可回退、可追踪。

## 与其他分支的关系
- **main**：当前所在稳定分支，作为项目的基准版本
- **experiment**：实验分支，用于测试/试验新功能；成熟后再合入 main

## main 分支目标
在此分支上维护稳定版本，接收已经验证过的功能、性能优化、修复和文档更新。进入 main 的改动应尽量通过类型检查和基础测试。

## 开发方式
- 由 Codex / Claude Code / 其他 AI Agent 协作开发
- AI 需要阅读 `CLAUDE.md` 了解当前分支上下文
- 如需试验高风险重构或破坏性改动，优先在 `experiment` 分支完成验证后再合入
